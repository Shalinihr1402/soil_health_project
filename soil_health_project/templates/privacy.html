Privacy Policy

Last updated: 2026

SoilSense is a student/demo project. This page explains, in plain terms, what data the app stores and why.

1. What We Store

  • Your name, email, and username, plus a securely hashed password (never stored in plain text).
  • Your soil analysis history: the nutrient values you enter and the resulting predictions.
  • If you use the city lookup, the city name is sent to OpenWeatherMap to fetch temperature and humidity.

2. How Data Is Used

Your data is used only to run the app's own features: showing your history, dashboard stats, and generating reports. It is not sold, shared with advertisers, or used for any purpose beyond this application.

3. Where Data Lives

All data is stored in a local SQLite database on the server running this application. There is no third-party analytics or tracking.

4. Your Choices

You can update your profile details or password at any time from Settings. Since this is a demo project, there is currently no self-service account deletion — contact whoever is running the instance if you'd like your data removed.
